# AOSP

> AOSP 全称 Android Open Source Project 是一个由 Google 领导的开源项目，旨在开发和维护 Android 操作系统的源代码。AOSP 提供了 Android 操作系统的基础框架和核心组件。本类目通过细读 AOSP 的源代码，深入理解 Android 操作系统的设计和实现原理。通过分析 AOSP 的代码结构、核心模块和关键功能，我们可以更好地理解 Android 的工作机制，为后续的逆向分析和安全研究打下坚实的基础。

!!! note 须知
    - 除非特别说明，本类目选取 Android 16.0.0_r2 版本的 AOSP 作为分析对象，此版本是落笔时最新的稳定版本。
    - 本类目按照 Android 中不同的核心模块进行划分，因为是学习笔记，划分不一定是最优的，内容可能也不一定完全详尽。
